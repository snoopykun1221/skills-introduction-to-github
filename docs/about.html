サイトについて

「ニッポン マップ ガイド」は、TikTok や Instagram などの SNS で話題になった日本各地のおすすめスポット・グルメ・神社・体験を、47 都道府県別にまとめたガイドサイトです。

SNS で気になったお店や場所は、フィードを流れていくうちにすぐ見つけられなくなってしまいます。当サイトは、そんな「気になる」を地図上に整理して、いつでも旅のしおりとして開ける場所を目指しています。

コンセプト

  • 地図から探す: 行きたい場所が決まっていなくても、都道府県をタップするだけで発見できる
  • SNS発の最新情報: TikTok・Instagram で話題のスポットを優先的にピックアップ
  • 4つのカテゴリ: 料理屋・行先・神社・体験で、旅の目的別に検索可能
  • 予約までシームレス: 各スポットから直接予約サイトへリンク

掲載基準

掲載しているスポットは、以下の基準で選定しています。

  • SNS(TikTok / Instagram / X)で一定の反応を得ている
  • 口コミサイト・レビューで高評価を得ている
  • 運営者または信頼できる情報源が訪問・確認している

免責・アフィリエイトについて

掲載情報は変更される可能性があります。実際にご利用される前に、各店舗・施設の公式情報を必ずご確認ください。当サイトは複数のアフィリエイトプログラムに参加しています。詳細は プライバシーポリシー をご覧ください。
